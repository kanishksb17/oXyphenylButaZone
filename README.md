# oXyphenylButaZone

## Setup

### 1. Install Python dependencies
```bash
pip install -r requirements.txt
```

### 2. Start the backend
```bash
python backend/app.py
```

### 3. Start the frontend
```bash
streamlit run frontend/app.py
```

## Structure

- `backend/`: Flask API server.
- `frontend/`: Streamlit web app.
